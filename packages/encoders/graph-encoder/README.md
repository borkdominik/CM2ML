# @cm2ml/graph-encoder

This plugin provides a raw graph encoder that supports feature encoding and both adjacency lists and matrices.
Based on the work of [López et al.](https://dl.acm.org/doi/10.1145/3550355.3552461)

## Development

### Installation

```bash
pnpm install
```

### Build

```bash
pnpm build
```

### Test

```bash
pnpm test
```

### Lint

```bash
pnpm lint
```

### Typecheck

```bash
pnpm check:tsc
```
