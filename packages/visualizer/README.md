# @cm2ml/visualizer

This package contains the visualizer of the CM2ML framework.

## Development

### Installation

```bash
pnpm install
```

### Build

```bash
pnpm build
```

### Test

```bash
pnpm e2e:prepare && pnpm e2e
```

### Lint

```bash
pnpm lint
```

### Typecheck

```bash
pnpm check:tsc
```
