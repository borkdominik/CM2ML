# @cm2ml/cli

This package provides a command `cm2ml` that can be used to run the CM2ML framework from the command line.
This CLI has all built-in parsers and encoders applied.

## Development

### Installation

```bash
pnpm install
```

### Build

```bash
pnpm build
```

### Lint

```bash
pnpm lint
```

### Typecheck

```bash
pnpm check:tsc
```
