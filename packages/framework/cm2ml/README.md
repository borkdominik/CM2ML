# @cm2ml/cm2ml

This package provides all built-in parsers and encoders of the CM2ML framework, as well as a CLI and REST server.

## Development

### Installation

```bash
pnpm install
```

### Build

```bash
pnpm build
```

### Lint

```bash
pnpm lint
```

### Typecheck

```bash
pnpm check:tsc
```
