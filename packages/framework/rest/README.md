# @cm2ml/rest

This package provides a command `cm2ml-server` that can be used to start a REST server that runs the CM2ML framework.
This server has all built-in parsers and encoders applied.

## Development

### Installation

```bash
pnpm install
```

### Build

```bash
pnpm build
```

### Lint

```bash
pnpm lint
```

### Typecheck

```bash
pnpm check:tsc
```
