# prefixspan

> TypeScript implementation of PrefixSpan algorithm, based on [PrefixSpan-py](https://github.com/chuanconggao/PrefixSpan-py).

## Development

### Installation

```bash
pnpm install
```

### Build

```bash
pnpm build
```

### Test

```bash
pnpm test
```

### Lint

```bash
pnpm lint
```

### Typecheck

```bash
pnpm check:tsc
```
