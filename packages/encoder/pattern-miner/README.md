# @cm2ml/pattern-miner

This package provides a pattern miner for the CM2ML framework.
Based on the work of [Fumagalli et al.](https://link.springer.com/chapter/10.1007/978-3-031-17995-2_4)

## Development

### Installation

```bash
pnpm install
```

### Build

```bash
pnpm build
```

### Test

```bash
pnpm test
```

### Lint

```bash
pnpm lint
```

### Typecheck

```bash
pnpm check:tsc
```
